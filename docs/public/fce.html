Fundamentals of Computer Engineering

Class Overview

This class provides a global vision of Computer Engineering from both academic and professional perspectives. It introduces the theoretical foundations of the field while also helping students develop practical skills, particularly in web development.

The subject is divided into two modules: Theoretical Foundations and Practical Skills.

Theoretical Foundations

  • Computer Engineering in Society: History, role of engineers, ethical and anthropological foundations.
  • Information and Data: Representation, data organization, number systems, codes, and data ethics.
  • Processing & Transmission: Systems, networks, current paradigms (Cloud, AI, IoT, Blockchain, etc.).

Practical Skills

  • Foundations of Software Development: Introduction to development environments, version control with Git, and collaborative work using GitHub.
  • HTML5: Structuring web pages, understanding the DOM, and creating basic forms.
  • CSS3: Styling web pages, working with layouts, the box model, and positioning elements.
  • GitHub: Use of the GitHub website and the GitBash commands.

The main practical project of the course consists of designing and developing a personal website using HTML and CSS, applying the concepts learned throughout the subject.

Objectives

  • Understand what computer engineering is and its impact on society.
  • Acquire a general historical perspective on the evolution of technology.
  • Identify the ethical and human aspects involved in engineering decisions.
  • Become familiar with current and emerging computing paradigms.
  • Explore possible professional paths within the field of computer engineering.

Learning Outcomes

  • Apply basic terminology related to computer engineering, both in hardware and software contexts.
  • Relate technological developments to their historical background and analyze their evolution.
  • Understand how information and data are represented and managed in computing systems.
  • Identify the main components and functions of modern computer systems.
  • Design and implement a simple informational website using HTML and CSS.
  • Communicate technical ideas clearly through written and oral presentations.
